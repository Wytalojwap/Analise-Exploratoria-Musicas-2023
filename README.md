# Análise Exploratória de Dados das Músicas Mais Tocadas em 2023
Este projeto tem como objetivo realizar uma análise exploratória de dados (EDA) a partir de um dataset contendo informações sobre as músicas mais tocadas em 2023. A análise foi conduzida com foco na compreensão do comportamento de consumo musical, identificação de padrões, tendências e possíveis oportunidades a partir dos dados.
